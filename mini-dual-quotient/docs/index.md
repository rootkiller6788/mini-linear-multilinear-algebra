# MiniDualQuotient Documentation

Welcome to the `mini-dual-quotient` package documentation.

## Overview

`mini-dual-quotient` provides dual spaces, double duals, transpose maps,
quotient spaces, annihilators, and the three isomorphism theorems for
finite-dimensional vector spaces.
